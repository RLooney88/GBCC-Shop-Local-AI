Shop Local Assistant - Embed Instructions

Important: Place this code where you want the chat widget to appear on your website.

Step 1: Add the following code to your website's HTML:

<div id="shop-local-assistant"></div>
<script>
  (function(w,d,s,o,f,js,fjs){
    w['ShopLocalWidget']=o;w[o]=w[o]||function(){
      (w[o].q=w[o].q||[]).push(arguments)};
    js=d.createElement(s),fjs=d.getElementsByTagName(s)[0];
    js.id='shop-local-assistant-script';
    js.src='REPLACE_WITH_YOUR_DOMAIN/widget.js';
    js.async=1;
    fjs.parentNode.insertBefore(js,fjs);
  }(window,document,'script','shopLocal'));
</script>

Step 2: Replace REPLACE_WITH_YOUR_DOMAIN

Replace REPLACE_WITH_YOUR_DOMAIN with your actual domain where the chat widget is hosted.

Example with domain replaced:

Note: Make sure to:

  • Replace your-domain.com with your actual domain
  • Keep the script tag at the bottom of your page for optimal loading
  • The widget will automatically handle styling and positioning
