Shop Local Assistant - Embed Instructions

Ready to use code: Copy and paste this code where you want the chat widget to appear on your website. 
<div id="shop-local-assistant"></div>
<script>
  (function(w,d,s,o,f,js,fjs){
    w['ShopLocalWidget']=o;w[o]=w[o]||function(){
      (w[o].q=w[o].q||[]).push(arguments)};
    js=d.createElement(s),fjs=d.getElementsByTagName(s)[0];
    js.id='shop-local-assistant-script';
    js.src='https://chamber.prmthe.us/widget.js';
    js.async=1;
    fjs.parentNode.insertBefore(js,fjs);
  }(window,document,'script','shopLocal'));
</script>

Important notes:

  • The widget will automatically position itself in the bottom-right corner of your website
  • The chat interface will maintain the chamber's branding (#00A7B7)
  • The widget is responsive and will work on both desktop and mobile devices
  • Place the code just before the closing </body> tag for optimal loading

Testing the Integration

After adding the code to your website:

  1. Clear your browser cache
  2. Refresh your webpage
  3. You should see a chat button in the bottom-right corner
  4. Click the button to open the chat interface
