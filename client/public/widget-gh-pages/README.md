## Setup Instructions
1. Create a new repository named `shop-local-widget` on GitHub
2. Clone the repository:
   ```bash
   git clone https://github.com/RLooney88/shop-local-widget.git
   ```
3. Copy these files into the repository:
   - `index.html`
   - `widget.js`
4. Push to GitHub:
   ```bash
   git add .
